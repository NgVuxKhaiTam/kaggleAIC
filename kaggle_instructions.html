For custom Ngrok domain, add your Ngrok auth token to Kaggle Secrets:

  • Add secret: NGROK_MAIN = your_ngrok_auth_token
Important Notes:
  • Make sure internet access is enabled in your Kaggle notebook
  • GPU should be enabled for model performance
  • Models will take 2-3 minutes to initialize on first run
  • The web interface will be accessible via the Ngrok URL shown in output
