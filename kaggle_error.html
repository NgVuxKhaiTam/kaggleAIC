Setup Error

There was a problem setting up the search system

Common Issues

  • Model Not Initialized: BEiT3 or SigLIP models haven't been loaded yet
  • File Not Found: Required files are missing from /kaggle/working/
  • GPU Memory: Insufficient GPU memory for model loading
  • Network Issues: Problems connecting to model repositories

Solutions

1. Check Model Status

Run this in a Kaggle cell to check model initialization:

# Check if models are initialized
print("BEiT3 initialized:", globals().get('is_initialized', lambda: False)())
print("SigLIP initialized:", globals().get('is_siglip_initialized', lambda: False)())
2. Restart and Re-run

If models aren't initialized:

  • Restart your Kaggle notebook
  • Make sure GPU is enabled
  • Re-run Cell 1 (backend_aic.py) completely
  • Wait for "Models initialized successfully" message
  • Then run Cell 2 (kaggle_web_backend.py)
3. Check File Structure

Ensure these files are in /kaggle/working/:

import os
files = os.listdir('/kaggle/working/')
required = ['kaggle_index.html', 'kaggle_web_backend.py']
for f in required:
    print(f"{f}: {'✓' if f in files else '✗'}")
